cmd Winget

Το winget είναι ένα εργαλείο γραμμής εντολών για την εγκατάσταση και διαχείριση εφαρμογών σε Windows, που αναπτύχθηκε από τη Microsoft. Το όνομά του προέρχεται από το "Windows Package Manager" και είναι μέρος της προσπάθειας της Microsoft να προσφέρει μια πιο ενσωματωμένη και απλή διαδικασία διαχείρισης πακέτων και εφαρμογών για τα Windows.

Εγκαταστήσετε νέες εφαρμογές

winget install [όνομα-εφαρμογής]

Αναζητήσετε διαθέσιμες εφαρμογές

winget search [όνομα-εφαρμογής]

Αφαιρέσετε εφαρμογές

winget uninstall [όνομα-εφαρμογής]

Λίστα Εγκατεστημένων Εφαρμογών

winget list

Πληροφορίες για Εφαρμογές

winget show [όνομα-εφαρμογής]

Δημιουργεί ένα αρχείο JSON που περιέχει όλες τις εγκατεστημένες εφαρμογές και τις εκδόσεις τους.

winget export [διαδρομή-αρχειου.json]

Επαναφέρει τις εφαρμογές που περιγράφονται σε ένα αρχείο JSON.

winget import [διαδρομή-αρχειου.jso]

Σύγκριση Εφαρμογών

winget compare [όνομα-εφαρμογής1] [όνομα-εφαρμογής2]

Ρυθμίσεις

winget settings
