Εντολές Συστήματος

Σύστημα - Αρχεία

Εμφανίζει λεπτομερείς πληροφορίες για το σύστημα, συμπεριλαμβανομένων των πληροφοριών του λειτουργικού συστήματος, του επεξεργαστή, της μνήμης και άλλων στοιχείων.

systeminfo

Εμφανίζει το παράθυρο διαμόρφωσης του συστήματος, όπου μπορείτε να διαμορφώσετε τις ρυθμίσεις εκκίνησης του συστήματος.

msconfig

Ανοίγει το Registry Editor, που σας επιτρέπει να τροποποιήσετε το Μητρώο των Windows, όπου αποθηκεύονται οι ρυθμίσεις και οι παράμετροι του συστήματος.

regedit

Εκτελεί μια επισκευή συστήματος για να εντοπίσει και να επιδιορθώσει τυχόν αρχεία συστήματος που έχουν καταστραφεί.

sfc /scannow

Ανοίγει το παράθυρο διαμόρφωσης πολιτικής ομάδας του συστήματος, όπου μπορείτε να διαμορφώσετε τις πολιτικές του συστήματος.

gpedit.msc

Διεργασίες - Υπηρεσίες

Εμφανίζει τη λίστα των εκτελούμενων διεργασιών στο σύστημα, μαζί με λεπτομέρειες όπως το ID διεργασίας, την κατάσταση και την κατανάλωση πόρων.

tasklist

Τερματίζει μια εκτελούμενη διεργασία βάσει του ID διεργασίας ή του ονόματος.

taskkill

Εκτελεί το Windows Management Instrumentation Command-line (WMIC) και παρέχει προηγμένες λειτουργίες διαχείρισης των διεργασιών, όπως η εμφάνιση λεπτομερών πληροφοριών και η διαχείριση των προτεραιοτήτων των διεργασιών.

wmic process

Ανοίγει το παράθυρο Services, όπου μπορείτε να διαχειριστείτε τις υπηρεσίες που εκτελούνται στον υπολογιστή σας. Μπορείτε να ενεργοποιήσετε, απενεργοποιήσετε ή αλλάξετε τις ρυθμίσεις των υπηρεσιών.

services.msc

Υλικό

Ελέγχει την κατάσταση του σκληρού δίσκου και επιδιορθώνει τυχόν σφάλματα αρχείων.

chkdsk

Εκτελεί τον εργαλείο αποκατάστασης συστήματος αρχείων (Disk Defragmenter) για τη βελτιστοποίηση της τοποθέτησης των αρχείων στον σκληρό δίσκο.

defrag

Εκκινεί το DiskPart, ένα εργαλείο γραμμής εντολών που σας επιτρέπει να διαχειρίζεστε τους σκληρούς δίσκους, τις κατατμήσεις και τους τόμους. Μπορείτε να δημιουργήσετε, να διαγράψετε, να μεταβάλετε το μέγεθος και να διαμορφώσετε κατατμήσεις.

diskpart

Ανοίγει το παράθυρο Disk Management, όπου μπορείτε να διαχειριστείτε τους δίσκους, τις κατατμήσεις και τους τόμους του συστήματος. Μπορείτε να δημιουργήσετε, να διαγράψετε, να αντιγράψετε, να μεταβάλετε το μέγεθος και να διαμορφώσετε τις κατατμήσεις.

diskmgmt.msc
