Αγγλικά Ελληνικά Περιγραφή
Command PromptΓραμμή ΕντολώνΗ Γραμμή Εντολών είναι ένα εργαλείο που επιτρέπει την εκτέλεση εντολών μέσω κειμένου στο περιβάλλον των Windows.
PowerShellPowerShellΤο PowerShell είναι ένα ισχυρό εργαλείο γραμμής εντολών και scripting για τη διαχείριση και αυτοματοποίηση των Windows.
RegistryΜητρώοΤο Μητρώο είναι μια βάση δεδομένων που αποθηκεύει ρυθμίσεις και παραμέτρους για το λειτουργικό σύστημα και τις εφαρμογές.
Task ManagerΔιαχειριστής ΕργασιώνΟ Διαχειριστής Εργασιών είναι ένα εργαλείο που εμφανίζει τις τρέχουσες διαδικασίες και την κατανάλωση πόρων του συστήματος.
ServiceΥπηρεσίαΗ Υπηρεσία είναι μια διαδικασία που εκτελείται στο παρασκήνιο και παρέχει λειτουργίες στο λειτουργικό σύστημα ή άλλες εφαρμογές.
DriverΟδηγόςΟ Οδηγός είναι ένα πρόγραμμα που επιτρέπει στο λειτουργικό σύστημα να επικοινωνεί με το υλικό του υπολογιστή.
Control PanelΠίνακας ΕλέγχουΟ Πίνακας Ελέγχου είναι μια σουίτα εργαλείων που χρησιμοποιούνται για την αλλαγή ρυθμίσεων και την παραμετροποίηση του συστήματος.
SettingsΡυθμίσειςΟι Ρυθμίσεις είναι μια εφαρμογή που παρέχει πρόσβαση σε διάφορες επιλογές και ρυθμίσεις του λειτουργικού συστήματος και των εφαρμογών.
Windows ExplorerΕξερευνητής των WindowsΟ Εξερευνητής των Windows είναι το εργαλείο που χρησιμοποιείται για την πλοήγηση και διαχείριση των αρχείων και φακέλων στο σύστημα.
File SystemΣύστημα ΑρχείωνΤο Σύστημα Αρχείων είναι η δομή που χρησιμοποιείται για την αποθήκευση και οργάνωση αρχείων στον υπολογιστή.
NTFSNTFSΤο NTFS (New Technology File System) είναι ένα σύστημα αρχείων που χρησιμοποιείται από τα Windows για την αποθήκευση και διαχείριση δεδομένων.
FAT32FAT32Το FAT32 (File Allocation Table 32) είναι ένα παλαιότερο σύστημα αρχείων που χρησιμοποιείται για αποθήκευση δεδομένων, αλλά έχει περιορισμούς σε μέγεθος αρχείων και δίσκων.
BitLockerBitLockerΤο BitLocker είναι μια λειτουργία κρυπτογράφησης δίσκου των Windows που παρέχει προστασία των δεδομένων μέσω κρυπτογράφησης της μονάδας δίσκου.
Group PolicyΟμάδα ΠολιτικήςΗ Ομάδα Πολιτικής είναι ένα σύστημα που επιτρέπει τη διαχείριση και την εφαρμογή ρυθμίσεων και περιορισμών για τους χρήστες και τους υπολογιστές σε ένα δίκτυο.
Active DirectoryActive DirectoryΤο Active Directory είναι μια υπηρεσία καταλόγου που διαχειρίζεται χρήστες, υπολογιστές και άλλες πόρους σε δίκτυα Windows.
DomainΤομέαςΟ Τομέας είναι μια ομάδα υπολογιστών και πόρων που διαχειρίζονται κεντρικά μέσω του Active Directory.
WorkgroupΟμάδα ΕργασίαςΗ Ομάδα Εργασίας είναι μια ομάδα υπολογιστών που συνδέονται τοπικά και μοιράζονται πόρους, αλλά δεν διαχειρίζονται κεντρικά.
Windows UpdateΕνημέρωση των WindowsΗ Ενημέρωση των Windows είναι η διαδικασία λήψης και εγκατάστασης των τελευταίων ενημερώσεων και διορθώσεων για το λειτουργικό σύστημα.
User Account Control (UAC)Έλεγχος Λογαριασμού Χρήστη (UAC)Ο Έλεγχος Λογαριασμού Χρήστη είναι μια λειτουργία που βοηθά στην πρόληψη μη εξουσιοδοτημένων αλλαγών στο σύστημα μέσω επιβεβαίωσης από τον χρήστη.
Virtual MachineΕικονική ΜηχανήΗ Εικονική Μηχανή είναι ένα λογισμικό που μιμείται έναν φυσικό υπολογιστή, επιτρέποντας την εκτέλεση πολλών λειτουργικών συστημάτων σε έναν υπολογιστή.
Hyper-VHyper-VΤο Hyper-V είναι μια τεχνολογία εικονικοποίησης της Microsoft που επιτρέπει τη δημιουργία και διαχείριση εικονικών μηχανών.
Remote DesktopΑπομακρυσμένη Επιφάνεια ΕργασίαςΗ Απομακρυσμένη Επιφάνεια Εργασίας επιτρέπει την πρόσβαση και έλεγχο ενός υπολογιστή από μια άλλη τοποθεσία μέσω δικτύου.
SnapshotΣτιγμιότυποΤο στιγμιότυπο είναι μια καταγραφή της κατάστασης ενός συστήματος ή εικονικής μηχανής σε μια συγκεκριμένη χρονική στιγμή.
Task SchedulerΠρογραμματιστής ΕργασιώνΟ Προγραμματιστής Εργασιών είναι ένα εργαλείο που επιτρέπει την αυτόματη εκτέλεση εντολών ή προγραμμάτων σε καθορισμένα χρονικά διαστήματα.
Event ViewerΕπιθεωρητής ΣυμβάντωνΟ Επιθεωρητής Συμβάντων είναι ένα εργαλείο που παρακολουθεί και καταγράφει γεγονότα και σφάλματα που συμβαίνουν στο σύστημα.
System RestoreΑποκατάσταση ΣυστήματοςΗ Αποκατάσταση Συστήματος είναι μια λειτουργία που επιτρέπει την επιστροφή του συστήματος σε μια προηγούμενη κατάσταση αν παρουσιαστούν προβλήματα.
Blue Screen of Death (BSOD)Μπλε Οθόνη Θανάτου (BSOD)Η Μπλε Οθόνη Θανάτου είναι μια οθόνη σφάλματος που εμφανίζεται όταν το λειτουργικό σύστημα των Windows αντιμετωπίζει σοβαρό σφάλμα και δεν μπορεί να συνεχίσει.
Driver UpdateΕνημέρωση ΟδηγούΗ Ενημέρωση Οδηγού είναι η διαδικασία λήψης και εγκατάστασης των τελευταίων εκδόσεων των οδηγών υλικού για βελτίωση της απόδοσης και της συμβατότητας.
Windows DefenderWindows DefenderΟ Windows Defender είναι ένα ενσωματωμένο εργαλείο προστασίας κατά ιών και κακόβουλου λογισμικού στα Windows.
DirectXDirectXΤο DirectX είναι μια συλλογή API της Microsoft για τη διαχείριση γραφικών, ήχου και εισόδου/εξόδου σε εφαρμογές πολυμέσων και παιχνίδια.
Windows StoreΚατάστημα WindowsΤο Κατάστημα Windows είναι μια πλατφόρμα από την οποία οι χρήστες μπορούν να κατεβάσουν και να εγκαταστήσουν εφαρμογές για τα Windows.
Windows SandboxWindows SandboxΤο Windows Sandbox είναι ένα εργαλείο που παρέχει έναν απομονωμένο και ασφαλή χώρο για την εκτέλεση εφαρμογών ή αρχείων χωρίς κίνδυνο για το κύριο σύστημα.
Hyper-V ManagerΔιαχειριστής Hyper-VΟ Διαχειριστής Hyper-V είναι ένα εργαλείο που επιτρέπει τη δημιουργία, ρύθμιση και διαχείριση εικονικών μηχανών χρησιμοποιώντας την τεχνολογία Hyper-V.
Remote Desktop ConnectionΣύνδεση Απομακρυσμένης Επιφάνειας ΕργασίαςΗ Σύνδεση Απομακρυσμένης Επιφάνειας Εργασίας είναι μια δυνατότητα που επιτρέπει την απομακρυσμένη πρόσβαση σε έναν υπολογιστή μέσω δικτύου.
System InformationΠληροφορίες ΣυστήματοςΗ Λειτουργία Πληροφορίες Συστήματος παρέχει λεπτομερείς πληροφορίες για το υλικό και το λογισμικό του υπολογιστή.
App DataΔεδομένα ΕφαρμογήςΤα Δεδομένα Εφαρμογής είναι ένας φάκελος που αποθηκεύει ρυθμίσεις και δεδομένα που σχετίζονται με τις εφαρμογές του χρήστη.
System Configuration (msconfig)Ρύθμιση Συστήματος (msconfig)Η Ρύθμιση Συστήματος είναι ένα εργαλείο που επιτρέπει τη διαχείριση των διαδικασιών εκκίνησης και των υπηρεσιών στο σύστημα.
File ExplorerΕξερευνητής ΑρχείωνΟ Εξερευνητής Αρχείων είναι το εργαλείο που χρησιμοποιείται για την πλοήγηση, αναζήτηση και διαχείριση αρχείων και φακέλων στον υπολογιστή.
Virtual DesktopΕικονική Επιφάνεια ΕργασίαςΗ Εικονική Επιφάνεια Εργασίας είναι μια λειτουργία που επιτρέπει τη δημιουργία και διαχείριση πολλών εικονικών επιφανειών εργασίας για την οργάνωση των εφαρμογών και των ανοιχτών παραθύρων.
Windows Defender FirewallΤείχος Προστασίας Windows DefenderΤο Τείχος Προστασίας Windows Defender είναι μια ενσωματωμένη προστασία που ελέγχει και επιτρέπει ή αποκλείει την κυκλοφορία του δικτύου στο σύστημα.
ClipboardΠρόχειροΤο Πρόχειρο είναι μια προσωρινή αποθηκευτική περιοχή που χρησιμοποιείται για την αποθήκευση δεδομένων που έχουν αντιγραφεί ή αποκοπεί, μέχρι να επικολληθούν σε άλλη θέση.
TaskbarΓραμμή ΕργαλείωνΗ Γραμμή Εργαλείων είναι η γραμμή στο κάτω μέρος της οθόνης που παρέχει πρόσβαση σε ανοιχτές εφαρμογές, εργαλεία και ρυθμίσεις του συστήματος.
Startup FolderΦάκελος ΕκκίνησηςΟ Φάκελος Εκκίνησης είναι ο φάκελος στον οποίο μπορείτε να τοποθετήσετε συντομεύσεις για εφαρμογές που θα εκκινούν αυτόματα κατά την εκκίνηση των Windows.
Credential ManagerΔιαχειριστής ΔιαπιστευτηρίωνΟ Διαχειριστής Διαπιστευτηρίων είναι ένα εργαλείο που αποθηκεύει και διαχειρίζεται διαπιστευτήρια, όπως ονόματα χρήστη και κωδικούς πρόσβασης, για διάφορες υπηρεσίες και εφαρμογές.
Safe ModeΑσφαλής ΛειτουργίαΗ Ασφαλής Λειτουργία είναι μια λειτουργία εκκίνησης που φορτώνει μόνο τα βασικά αρχεία και οδηγούς, βοηθώντας στη διάγνωση και επίλυση προβλημάτων.
Event IDID ΣυμβάντοςΤο ID Συμβάντος είναι ένας αριθμητικός κωδικός που αναγνωρίζει συγκεκριμένα συμβάντα καταγραφής στο Σύστημα.
Windows Subsystem for Linux (WSL)Υποσύστημα Windows για Linux (WSL)Το Υποσύστημα Windows για Linux (WSL) επιτρέπει την εκτέλεση περιβαλλόντων Linux και εργαλείων απευθείας στα Windows χωρίς τη χρήση εικονικής μηχανής.

Windows BackupΑντίγραφα Ασφαλείας των WindowsΗ Λειτουργία Αντίγραφα Ασφαλείας των Windows είναι μια υπηρεσία που επιτρέπει τη δημιουργία αντιγράφων ασφαλείας των αρχείων και των ρυθμίσεων του συστήματος.
