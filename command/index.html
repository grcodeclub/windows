Γραμμή Εντολών

Η γραμμή εντολών (Command Prompt) στα Windows είναι μια εφαρμογή που επιτρέπει στους χρήστες να εκτελούν εντολές και να διαχειρίζονται το σύστημα τους μέσω κειμένου. Μπορεί να χρησιμοποιηθεί για να εκτελέσεις διάφορες λειτουργίες στον υπολογιστή σου, όπως τη δημιουργία, τη μετονομασία, τη μετακίνηση ή τη διαγραφή αρχείων, την εκκίνηση προγραμμάτων, τη διαχείριση των ρυθμίσεων του συστήματος και πολλά άλλα.
