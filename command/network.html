Εντολές Δικτύου

ipconfig

Εμφανίζει πληροφορίες για τις δικτυακές συνδέσεις και διευθύνσεις IP.

ipconfig

Εμφανίζει λεπτομερείς πληροφορίες για όλες τις δικτυακές συνδέσεις, συμπεριλαμβανομένων των διευθύνσεων MAC και διαμορφωμένων ρυθμίσεων DNS.

ipconfig /all

Καθαρίζει τη μνήμη cache του DNS, κάτι που μπορεί να βοηθήσει σε προβλήματα προσβασιμότητας ιστοσελίδων.

ipconfig /flushdns

Εμφανίζει τα καταχωρημένα DNS από τη μνήμη cache.

ipconfig /displaydns

Απελευθερώνουν και ανανεώνουν τη διεύθυνση IP αντίστοιχα..

ipconfig /release
ipconfig /renew

ping

Αποστέλλει πακέτα ICMP σε έναν υπολογιστή για έλεγχο σύνδεσης.

ping

tracert

Εμφανίζει τη διαδρομή που ακολουθούν τα πακέτα για να φτάσουν σε έναν προορισμό.

tracert

netstat

Παρέχει πληροφορίες για τις δικτυακές συνδέσεις.

netstat

Εμφανίζει όλες τις διαθέσιμες δικτυακές συνδέσεις και τις ακούσιες πόρτες.

netstat -a

Εμφανίζει τις διευθύνσεις IP και τις αριθμημένες πόρτες αντί των ονομασιών κειμένου.

netstat -n

Εμφανίζει τη λίστα των εφαρμογών που συνδέονται με το δίκτυο και τις διαθέσιμες συνδέσεις.

netstat -b

Εμφανίζει τις δικτυακές συνδέσεις μόνο για το συγκεκριμένο πρωτόκολλο (π.χ. tcp ή udp).

netstat -p <protocol⪫

Εμφανίζει τον πίνακα δρομολόγησης.

netstat -r

net

Διαχειρίζεται τους χρήστες του δικτύου, συμπεριλαμβανομένης της αλλαγής κωδικών.

net user

Διαχειρίζεται τις ομάδες χρηστών του δικτύου.

net group

Εμφανίζει τους διαθέσιμους υπολογιστές και κοινόχρηστους φακέλους στο δίκτυο.

net view

Χρησιμοποιείται για τη σύνδεση και αποσύνδεση κοινόχρηστων και δικτυακών συσκευών.

net use

Διαφορά

Ανοίγει το παράθυρο των δικτυακών συνδέσεων για να διαμορφώσετε τις ρυθμίσεις του δικτύου.

ncpa.cpl
