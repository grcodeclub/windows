PowerShell - Αναφορές

Εξαγωγή Αναφορών

Εάν χρειάζεται να εξαγάγετε δεδομένα που μπορείτε να χρησιμοποιήσετε σε μια αναφορά ή να στείλετε σε κάποιον άλλο, το ConvertTo-HTML είναι ένας απλός τρόπος για να το κάνετε. Για να το χρησιμοποιήσετε, διοχετεύστε την έξοδο από μια άλλη εντολή στην εντολή ConvertTo-HTML και χρησιμοποιήστε το διακόπτη - Ιδιότητα για να καθορίσετε ποιες διότητες εξόδου θέλετε στο αρχείο HTML. Θα χρειαστεί επίσης να δώσετε ένα όνομα αρχείου.

Για παράδειγμα, ο ακόλουθος κώδικας δημιουργεί μια σελίδα HTML που παραθέτει τα ψευδώνυμα PowerShell στην τρέχουσα κονσόλα:

get-alias | convertto-html > aliases.html
