PowerShell | Yπηρεσίες

Get-Service

Δείτε ποιες υπηρεσίες είναι εγκατεστημένες στον υπολογιστή σας.

Get-Service

Επιστρέφει πληροφορίες για μια συγκεκριμένη υπηρεσία με βάση το όνομά της.

Get-Service -Name Service_Name_Here

Επιστρέφει πληροφορίες για μια υπηρεσία με βάση την εμφανιζόμενη ετικέτα της.

Get-Service -DisplayName Display_Name_Here

Επιστρέφει τις υπηρεσίες που βρίσκονται σε κατάσταση εκτέλεσης.

Get-Service | Where-Object { $_.Status -eq "Running" }

Επιστρέφει τις υπηρεσίες ταξινομημένες κατά αλφαβητική σειρά βάσει της εμφανιζόμενης ετικέτα

Get-Service | Sort-Object DisplayName

Start-Service

Ξεκινά μια υπηρεσία που έχει διακοπεί.

Start-Service

Αυτή η εντολή θα επιλέξει όλες τις υπηρεσίες που έχουν κατάσταση "Stopped" και στη συνέχεια θα τις ξεκινήσει χρησιμοποιώντας την εντολή Start-Service.

Get-Service | Where-Object { $_.Status -eq "Stopped" } | Start-Service

Stop-Service

Διακόπτει μια υπηρεσία που εκτελείται.

Stop-Service

Restart-Service

Επανεκκινεί μια υπηρεσία.

Restart-Service

Set-Service

Επιτρέπει την τροποποίηση των ρυθμίσεων μιας υπηρεσίας.

Set-Service

Αυτό θα ορίσει την υπηρεσία με το όνομα "MyService" να ξεκινά αυτόματα κατά την εκκίνηση του συστήματος.

Set-Service -Name "MyService" -StartupType Automatic

Αυτό θα ορίσει την υπηρεσία με το όνομα "MyService" να εκτελείται υπό τον λογαριασμό που ορίζεται στη μεταβλητή $cred.

Set-Service -Name "MyService" -StartupType Automatic -Credential $cred

Αυτό θα ορίσει την περιγραφή της υπηρεσίας "MyService" σε "My Custom Service".

Set-Service -Name "MyService" -Description "My Custom Service"

New-Service

Δημιουργεί μια νέα υπηρεσία.

New-Service

Remove-Service

Αφαιρεί μια υπηρεσία από το σύστημα.

Remove-Service
